## Single Prompt vs Conversational Flow

The editor layout differs slightly between agent types:

| Element | Single Prompt | Conversational Flow |
|---------|---------------|---------------------|
| Top area | Prompt Section (Model, Voice, Language) | Toggle tabs (Workflow / Settings) |
| Center | Prompt editor | Visual canvas |
| Right sidebar | Configuration Panel | Configuration Panel (when in Settings) |
| Left sidebar | Same | Same |

In Conversational Flow, click **"Settings"** tab to access the same configuration options available in Single Prompt.

## Configuration Areas in Detail

### Prompt Section

Located in the top bar, provides quick access to:
- **Model** ‚Äî Select LLM (GPT-4o, etc.)
- **Voice** ‚Äî Choose from voice library
- **Language** ‚Äî Set primary language

‚Üí See: [Model Selection](/platform/configuring-agents/prompt-section/model-selection), [Voice Selection](/platform/configuring-agents/prompt-section/voice-selection), [Language Selection](/platform/configuring-agents/prompt-section/language-selection)

### Configuration Panel

Located in the right sidebar, provides toggles for:
- **End Call** ‚Äî When/how to terminate calls
- **Transfer Call** ‚Äî Handoff to humans
- **Knowledge Base** ‚Äî Attach reference docs
- **Variables** ‚Äî Dynamic values in prompts
- **API Calls** ‚Äî External service connections

‚Üí See: [End Call](/platform/configuring-agents/configuration-panel/end-call), [Transfer Call](/platform/configuring-agents/configuration-panel/transfer-call), [Knowledge Base](/platform/configuring-agents/configuration-panel/knowledge-base), [Variables](/platform/configuring-agents/configuration-panel/variables), [API Calls](/platform/configuring-agents/configuration-panel/api-calls)

### Agent Settings

Accessed via left sidebar, contains detailed tabs:
- **Voice Tab** ‚Äî Speech settings, pronunciation, advanced voice
- **Model Tab** ‚Äî LLM settings, language switching
- **Phone Number Tab** ‚Äî Assign phone number
- **Webhook Tab** ‚Äî Event notifications
- **General Tab** ‚Äî Timeout settings

‚Üí See: [Voice Tab](/platform/configuring-agents/agent-settings/voice-tab), [Model Tab](/platform/configuring-agents/agent-settings/model-tab), [Phone Number Tab](/platform/configuring-agents/agent-settings/phone-number-tab), [Webhook Tab](/platform/configuring-agents/agent-settings/webhook-tab), [General Tab](/platform/configuring-agents/agent-settings/general-tab)

## Saving Changes

Most changes save automatically. However, some sections show:

```
"You have unsaved changes"
[Discard] [Save]
```

Always click **Save** before leaving if you see this message.
